# timeGO

一款 macOS 平台的计时提醒 app，力求简洁快速

## 简介

- 可配置时间和通知信息，可组合作为番茄钟
- 状态栏小工具，轻量
- 设计简洁，适应系统主题，Dark 模式显示更佳
